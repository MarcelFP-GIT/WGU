2.2 Basic HTML Tags

Sections

Related content should be split into section using the <section> tag. Different heading levels should be used to organize sub-sections.

Headings

Headings are defined with <h1 - h6>Heading Text</h1-h6>.

Only empty elements do not need a closing tag.

Paragraphs

Paragraphs are defined with <pParagraph Text. HTML </p>.
